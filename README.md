# Docker

<pr>

## Docker command basic

```
docker --version
docker info
docker ps
docker ps -a
docker images
docker system df
```

## Docker build
```
docker pull <image_name>
docker build -t <Set image_name> .
```

## Docker run
```
docker start <NAME>
docker start <CONTAINER_ID>
docker stop <NAME>
docker stop <CONTAINER_ID>
```

## Docker danger zone
-f = force
```
docker rmi <IMAGE_ID>
docker rm <CONTAINER_ID>
docker <Type> prune 
```
